#  Estudos em Banco de Dados – SQL / MySQL

Este repositório reúne **scripts SQL desenvolvidos para estudo e prática de Banco de Dados**, utilizando **MySQL**.  
Os exemplos abrangem desde **modelagem de dados**, **criação de tabelas**, **relacionamentos**, **restrições de integridade**, até **consultas SQL avançadas**.

Este repositório reúne os projetos desenvolvidos no 2º semestre da graduação, na disciplina de Banco de Dados do curso de Desenvolvimento de Software Multiplataforma da FATEC de Registro, demonstrando a aplicação prática dos conceitos estudados ao longo do semestre.



##  Conceitos Abordados

- Modelagem de Banco de Dados Relacional
- Chaves Primárias e Estrangeiras
- Integridade de Entidade
- Integridade Referencial
- Integridade de Domínio
- Normalização
- Regras de Negócio no Banco
- SQL DDL e DML



##  Tecnologias Utilizadas

- **MySQL**
- **SQL**
- **MySQL Workbench** (ou similar)
